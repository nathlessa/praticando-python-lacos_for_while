# Praticando Python: Laços `for` e `while`

Este repositório contém exercícios desenvolvidos durante o curso **"Praticando Python: laços for e while"** da Alura.

## Sobre o projeto

Os exercícios têm como objetivo reforçar conceitos fundamentais de lógica de programação utilizando Python, com foco em estruturas de repetição.

Durante a prática, foram abordados:

- Uso do `for` para percorrer sequências
- Uso do `while` para controle de repetição com condições
- Estruturas condicionais (`if`, `else`, `continue`, `break`)
- Manipulação de listas e dicionários
- Validação de entradas com `input()` e `len()`
- Construção de lógicas como:
  - Contagem regressiva
  - Controle de estoque
  - Validação de cadastro de usuário

## Tecnologias utilizadas

- Python

## Objetivo

Desenvolver o raciocínio lógico e a capacidade de resolver problemas utilizando estruturas de repetição, além de consolidar boas práticas na escrita de código.
